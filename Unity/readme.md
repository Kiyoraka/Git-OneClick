## Unity Project Structure 📁

The script will properly manage the following Unity folders:

### Included in Git Repository ✅
- `/Assets`
- `/ProjectSettings`
- `/Packages`

### Ignored by Git ❌
- `/Library`
- `/Temp`
- `/Logs`
- `/UserSettings`
- `/obj`
- `/Build`
- `/Builds`
- Visual Studio / VS Code files
- Build files
- Temporary files
